## 功能亮点

- 当月预算 / 当日预算双输入
- 年历 / 月历视图切换
- 每日支出登记
- 自动计算当日结余与累计储备金
- 热力格展示每日预算执行情况
- 本地自动保存（`localStorage`）

## 本地运行

直接双击 `index.html` 即可打开。

## 部署到 GitHub Pages

1. 创建公开仓库并上传项目文件
2. 进入仓库 `Settings` -> `Pages`
3. 选择 `Deploy from a branch`
4. 选择 `main` + `/ (root)` 并保存
5. 等待 1-3 分钟后通过 Pages 链接访问

## 数据说明

- 当前版本为纯前端本地存储
- 不同用户 / 不同设备的数据彼此独立，不会自动同步
- 如需多人共享数据，需要接入后端数据库
